Ассортимент «Смарт - Билдер» - самое передовое в мире программное обеспечение для проектирования стеклянных изделий.

«Смарт-Билдер» - самое передовое в мире программное обеспечение для проектирования безрамных конструкций из стекла. Есть четыре модуля, которые можно использовать в качестве отдельных инструментов для проектирования или в качестве интегрированных в» систему продаж «Smart-Toolbox» («Смарт – Тулбокс»).

  • Smart-Shower
    («Смарт - Душ»)

    для конфигурирования душевых из стекла, включая кабины или перегородки для ванной.

    подробнее

    Smart-Shopfront
    («Смарт - Витрина»)

    для проектирования витрин из стекла или внутренних стеклянных перегородок.

  • Smart-Balustrade («Смарт - Ограждение»)

    для постройки структурированных ограждений из стекла или ограждений для бассейнов.

    подробнее

    Smart-Railing
    («Смарт - Перила»)

    для расчета опорных перил с кабелем или наполнителем из стекла.

    для проектирования стеклянных панелей под умывальной раковиной на кухне или в ванной.

    Наша прогрессивная система продаж

    Smart-Toolbox («Смарт-Тулбокс» - инструментальные средства разработки программ»)

    организует работу с клиентами и обсчитывает стоимость стекла (производимого или находящегося на складе), оснастки компьютерных работ и конструкций из любого конструкционного модуля, перечисленных выше. «Smart-Toolbox» содержит встроенную поддержку операционных действий, таких как заказы на проведение работ, заказы на покупку, а также легко выполнимые операции, такие, как «тащи» и «опускай», заранее запланированные мероприятия и установки плюс финансовые операции – выставление счетов, получение оплаты (выдача расписок в получении) и учет.
